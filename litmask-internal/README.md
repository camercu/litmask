# litmask-internal

Internal implementation crate for [`litmask`](https://crates.io/crates/litmask):
shared crypto primitives, wire-format constants, and pure helpers used by
`litmask`, `litmask-build`, and `litmask-macros`.

**Not a public API.** Do not depend on this crate directly — it carries no
stability guarantee and may change shape in any release. Depend on
[`litmask`](https://crates.io/crates/litmask) instead.
